Describe CSS selectors and list the types of selectors (e.g., element, class, id). ans= 1.element (type) selector: selects html elements by their tag name. ex: (in css) h1{ color:red; } 2.class selector: selects elements with a specific class attribute. use a dot(.) before the class name. ex(in html):
(in css): .box{ background-color:red; } 3.id selector : selects an element with a specific id attribute. use a hash(#) before the id name. ex (in html):
(in css): #header{ height:50px; } 4.universal selector: selects all elements on the page. ex: (in css) *{ margin:0; padding:0; } 5.group selector: applies the same styles to multiple selectors. ex:(in css) h1,h2,p{ color:red; font-family:Arial; }
